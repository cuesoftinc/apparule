# Demo photo attributions (mobile seed pool)

CC-licensed outfit photography rendered by the seeded `*Fake` repositories
(mobile-implementation.md §6) — byte-identical copies of the web mock
server's pool (`web/public/demo/ATTRIBUTIONS.md`, design.md §8.3 sourcing
decision: Openverse/Wikimedia Commons; same pool as the Figma Assets page),
so the phone and the dashboard render the same story over the same media.
Dev-flavor-scoped: these never ship in prod bundles.

All images are derivatives of the Commons originals: resized to ≤900 px on
the longest side and JPEG-recompressed (≤~200 KB); `outfit-w00.jpg` is
additionally rotated to its upright orientation.
